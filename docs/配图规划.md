# 配图规划

这份文档只保留“先画什么最有价值”，不再写模板化占位字段。  
统一存放路径仍建议使用 `docs/images/`。

**命名约定**：文件名里的 `ch1`、`ch2`、`ch3`、`ch5`、`ch9` 等为**章节序号简称**，与正文「第一章」「第二章」…「第九章」一一对应，不等同于仓库目录名 `chapter1/`。

## 优先补

1. `readme-learning-roadmap.png`：整个仓库的学习路线总览图（给新读者快速定向）。
2. `ch1-multimodal-map.png`：多模态系统主链路图（输入 → 编码器 → 对齐 → LLM → 输出）。
3. `ch2-patch-tokenization.png`：像素到 patch token 的过程图（解释“为什么会丢细节”）。
4. `ch3-vlm-architectures.png`：三种主流架构路线对照图（双塔、Connector + LLM、统一建模）。
5. `ch5-eval-matrix.png`：评测维度矩阵图（感知、OCR、推理、幻觉、工程成本）。
6. `ch9-agent-workflow.png`：多模态 Agent 工作流图（感知、规划、工具、记忆）。
